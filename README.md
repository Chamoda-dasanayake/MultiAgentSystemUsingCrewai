# Multi-Agent Stock Analysis System

This project is a **multi-agent stock analysis system** built using **CrewAI** and **Groq Llama models**. It demonstrates how AI agents can work together to fetch live stock data, analyze market trends, and provide actionable insights.

---

## Features

- **Analyst Agent:** Evaluates publicly traded stocks using live market data and financial indicators.  
- **Custom Stock Tool:** Retrieves current stock prices, daily changes, trading volume, and other market metrics.  
- **Live Data Analysis:** Provides summary insights and identifies immediate trends in stock performance.  
- **Agent Orchestration:** Demonstrates the use of AI agents for structured financial decision-making.

---

## Technologies Used

- Python 3.12  
- [CrewAI](https://www.crewai.com/) for agent orchestration  
- [Groq Llama Models](https://www.groq.com/) for LLM integration  
- [yfinance](https://pypi.org/project/yfinance/) for fetching live stock data  
- dotenv for managing API keys
